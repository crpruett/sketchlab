# Christopher Pruett

DevOps Engineer, Cloud Native Engineer, command line junkie. I work mainly with Linux, Azure, Kubernetes, and C++.

[I have a passion for Philosophical DevOps](https://ginraidevops.substack.com/p/the-point?r=3gxfuj) and love both learning and teaching. 

## Things I've Built

### Kubernetes Homelab

My pride and joy. Enterprise grade security in my basement with Talos Linux and FluxCD.



## Microsoft Azure

I work as a Cloud Engineer in an Azure environment and I've decided to go all-in on Azure and focus on that ecosystem. My employer kindly supplied me with a monthly credit which I can use for learning, so I'm building up an Azure Lab which I'm documenting on my blog and LinkedIn.

## C++

No matter what language I need to do some other task, I always end up coming back to my first love: C++. 

After a a few years learning the fundamentals with Java, I have returned to C++. It's the language I have most experience with, and which feels most natural to me, currently.

I enjoy that I can use it for practically everything and it's what I've learned the most with, and been able to build some cool things.
